# 📝 Notes App

A full-stack Notes App with user authentication, password recovery, and note management, built using React, Express.js, MongoDB, and JWT.

<img alt="Screenshot 2025-07-17 171320" src="https://github.com/user-attachments/assets/d7f8e185-1b6f-44ea-b664-a42cadd2a6ad" />


## 🚀 Features

- ✅ User registration & login with JWT authentication  
- 🔐 Protected routes using middleware  
- 🗃️ Create, read, update, and delete personal notes  
- 🔁 Forgot password flow with secure token reset  
- ☁️ Deployed on Render (Client + Server)

## 🛠️ Tech Stack

- **Frontend:** React, Axios, React Router  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Auth:** JWT, bcrypt  
- **Email:** nodemailer  
- **Deployment:** Render
